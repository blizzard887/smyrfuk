Алгебра

Основные формулы и правила:

  • Квадратное уравнение: ax² + bx + c = 0. Решение: x = (-b ± √D) / 2a, где D = b² - 4ac.
  • Теорема Виета: Для уравнения x² + px + q = 0, если x₁ и x₂ — корни, то x₁ + x₂ = -p и x₁ · x₂ = q.
  • Арифметическая прогрессия:
    • n-й член: aₙ = a₁ + (n - 1)d
    • Сумма первых n членов: Sₙ = (a₁ + aₙ) · n / 2
  • Геометрическая прогрессия:
    • n-й член: bₙ = b₁ · q^(n-1)
    • Сумма первых n членов: Sₙ = b₁ · (qⁿ - 1) / (q - 1)

Пример 1: Решите уравнение x² - 5x + 6 = 0.

Решение:

1. Находим дискриминант: D = (-5)² - 4 · 1 · 6 = 25 - 24 = 1.

2. Корни уравнения: x = (5 ± √1) / 2.

3. Получаем: x₁ = 3, x₂ = 2.

Ответ: x₁ = 3, x₂ = 2.

Пример 2: Найдите сумму первых 10 членов арифметической прогрессии, если a₁ = 2 и d = 3.

Решение:

1. Находим 10-й член: a₁₀ = a₁ + (10 - 1)d = 2 + 9 · 3 = 29.

2. Сумма первых 10 членов: S₁₀ = (a₁ + a₁₀) · 10 / 2 = (2 + 29) · 5 = 155.

Ответ: S₁₀ = 155.

Геометрия

Основные теоремы и формулы:

  • Теорема Пифагора: В прямоугольном треугольнике a² + b² = c², где c — гипотенуза.
  • Площадь треугольника: S = ½ · a · h, где a — основание, h — высота.
  • Площадь круга: S = πr², где r — радиус.

Пример 3: Найдите площадь круга с радиусом r = 5.

Решение:

1. Используем формулу: S = πr².

2. Подставляем значение: S = π · 5² = 25π.

Ответ: S = 25π.
